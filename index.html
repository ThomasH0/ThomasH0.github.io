Topic

CSS is one of the three biggest tools for web development. Obviously, the other two being HTML and Java Script. It was soely created to make websites look better and give the programmer an easier time to create and edit their websites to a tailored theme. CSS stands for Cascading Style Sheets, allowing users to use more than one style. This website itself uses CSS to create this theme and animation style. CSS shaped the internet to where it is today, without it who knows what the internet would look like.

Check the other sections for more information about CSS.

Features

CSS has many features and everything in CSS is based on boxes. You can have custom images, shapes, sizes, text, animations, etc. The great thing about CSS is how fully customizable it is. One page can have a set theme and the other a completely different theme. You are able to give certain pages, or HTML features an ID or CLASS tag that allows you in CSS to fully customize that specific tag. It comes down to even the most basic HTML features. You can have one paragraph with a certain text font and size and another with a completely different style based on the extension of tag.

CSS is great because it is very simple to write in. There are two types of syntax. Selector and Declaration. A selector in HTML would be a < p>< /p> tag and the declaration would be the CSS code in the curly bracket stating it's properties. That's the appealing part, it's easy to learn and work on. You can have certain selectors with specific declarations.

Another great feature about CSS is that there are 3 Ways to use CSS. You have External, Internal, and Inline

External: Allows you to change the CSS of multiple files from only one file. The CSS code is written in an external file that is linked to the HTML file.
Internal: Instead of all the CSS code being in an external file, it is all within the HTML code.
Inline: Is used to assign a unique element to an HTML page directly on the line. Inline has a higher priority over the other two.

History

CSS was first created in 1994 by a Norweigan man by the name of Håkon Wium Lie in an attempt to create a standard style sheet for the whole wide web to use. Obviously it was successful, and ushered in an era of new and improved websites with a very unique style that can be tailored and customed to each website, also allowing each website to use multiple themes depending on the page if so desired.

Who manages CSS Now?
CSS is now maintained by a group inside W3C called the CSS Working Group. They create document specifications where they come up with and discuss new features, if the board approves it becomes a recommendation. They are recommendations because W3C does not have control over the language, rather indepedent companies work on the software.
